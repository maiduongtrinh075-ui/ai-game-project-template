# Expand Module

When expanding an existing module:

1. Read `CLAUDE.md`.
2. Read the matching file in `docs/systems/`.
3. Read the local module README and source files.
4. Keep changes inside the owning module and its immediate UI or data dependencies.
5. Preserve external contracts unless the task explicitly says otherwise.
6. Implement the smallest useful expansion first.
7. Record lasting architectural changes in `.claude/memory/decisions.md`.
