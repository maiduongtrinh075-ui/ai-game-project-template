# Expand Module Workflow

1. Read `AI.md`.
2. Read `.ai/context/boundaries.md`.
3. Read `docs/systems/<module>.md`.
4. Read the target module README and source files.
5. Identify inputs, outputs, and do-not-break interfaces.
6. Implement the smallest useful expansion first.
7. Verify the module still connects cleanly to the rest of the project.
8. Record any lasting decisions in `.ai/memory/decisions.md`.
